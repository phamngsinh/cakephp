Croogo Theme: BsCroogo
========

A Bootstrap starter theme for Croogo 1.5 (a cms based on CakePHP)

Screenshot
--------------

![Alt text](/webroot/img/screenshot.png "thumb")

Features
--------------
- Two column layout
- Auto hide column when empty (no blocks inside)
- Bootstrap 2.3.1
- Font Awesome 3.1.0
